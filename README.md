# Media Store System

Ett system för att hantera produkter, lager och försäljning i en butik.

## Funktioner

* Lägga till, uppdatera och ta bort produkter
* Hantera lagerstatus
* Genomföra köp med varukorg
* Spara och läsa data från CSV-fil

## Tekniker

* C#
* .NET (Windows Forms)
* CSV-filhantering

## Min roll

Jag utvecklade applikationen själv och ansvarade för:

* Design av systemstruktur
* Implementering av funktioner
* Hantering av data och felhantering
* Separation av logik och användargränssnitt
